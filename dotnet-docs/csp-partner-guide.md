# Telerik Report Server for .NET — CSP Partner Guide

This page provides Cloud Solution Provider (CSP) partners with the essential information for reselling and supporting [Progress Telerik Report Server](https://www.telerik.com/report-server) for .NET through the Microsoft Azure Marketplace.

## About Telerik Report Server for .NET

Telerik Report Server for .NET is a self-hosted, server-based reporting platform that enables organizations to create, manage, schedule, and deliver reports through a centralized web-based interface. It is available on the [Azure Marketplace](https://marketplace.microsoft.com/en-us/product/progresssoftwarecorporation.progress-telerik-report-server) as a Bring Your Own License (BYOL) offering deployed on an Azure Virtual Machine.

For a full product overview, visit the [Telerik Report Server Homepage](https://www.telerik.com/report-server).

## Licensing

This is a **BYOL (Bring Your Own License)** offer:

- Customers purchase a Telerik Report Server license separately from [Progress Software](https://www.telerik.com/purchase/report-server).
- Azure infrastructure costs (VM, storage, networking) are billed through the customer's Azure subscription, which can be CSP-managed.

A [free trial license](https://www.telerik.com/account/trials) is available for evaluation.

## Support

Partners in the CSP program are expected to provide tier 1 support to their customers. For issues requiring escalation, submit a ticket through the [Telerik Support Portal](https://www.telerik.com/account/support-tickets).

For sales inquiries, [contact Progress Sales](https://www.telerik.com/contact).
