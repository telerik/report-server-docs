# System Requirements

## Server
- Windows OS

- Internet Information Services (IIS 7+)

- Microsoft .NET Framework 4.6.2+

## Client
- HTML5 compliant browser with support for JavaScript 1.8.5 (ECMAScript5) and CSS3
